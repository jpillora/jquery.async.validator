jQuery Async Validator

An jQuery plugin to provide easy to use asynchronous client side validations.

Github »

Basic Example

Given the following: 
<form id='myForm'>
  <input value='42'>
  <input value='10/06/2012'>
  <input class='submit' type='submit'/>
</form>
Add some validators to the inputs: 
<form id='myForm'>
  <input value='42' data-validate='number'>
  <input value='10/06/2012' data-validate='date'>
  <input class='submit' type='submit'/>
</form>
